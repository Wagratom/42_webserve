O Aplicativo:

Foi construído utilizando AWS Lambda, API Gateway, s3 e SQS.

Demonstra o uso de tecnologias serverless para gerenciamento de senhas.

Oferece um sistema de cadastro, onde o usuário envia suas senhas e verifica se atendem aos requisitos.

Encaminha a senha recebida para uma fila, onde será recebida por outra função Lambda que fará a validação.

Encaminha ao e-mail do usuário, informaçãoes sobre o status do cadastro.
